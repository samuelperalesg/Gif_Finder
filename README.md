### Trello Board: https://trello.com/invite/b/VE9WI5Kz/ATTI6ad8c4a631f2d09bc04676b436365f14D022C6A5/gifinder

## Technologies Used
* HTML5
* CSS3
* JavaScript
* jQuery
* Google Fonts
* Giphy API
